## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
# instalcion de dependencias nesesaria para la ejecucion del proyecto

$ pnpm i
```

## Prisma

```bash
# creamos la migracion de nuesto schema de prisma a las tablas de postgresql en este caso

$ pnpm run prisma:migrate

```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test | <smal>pending ...</small>

# unit tests

$ pnpm run test:cov

## enviroments

```bash
# url de la base de datos, en este caso apuntamos un contenedor de docker que levanta un db
DATABASE_URL="postgresql://root:root@0.0.0.0:5432/root"

# string secreat para el hash del token JWT
JWT_SECRET="examplejwtsecret"

# duracion de la utilidad de nuesto token de acceso
JWT_EXPIRES_IN="3d"

# nivel de complejidad de hash de muesta password
HASH_SALT=10
```
